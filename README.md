## 🍃 동물의 숲 키보드 사운드 프로그램 (Windows)

> **키보드를 입력할 때마다 동물의 숲 감성 사운드가 재생됩니다!**
> 본 프로젝트는
> 👉 [joeyycho님의 동물의 숲 키보드 프로젝트](https://github.com/joeyycho/keyboardsounderforwin.git)를
> **기반으로 리디자인 및 기능을 개선한 Windows용 프로그램**입니다.

---

## ✨ 주요 특징

* ⌨️ **키 입력 시 동물의 숲 스타일 사운드 재생**
* 🎨 **동물의 숲 테마 UI로 전면 리디자인**
* 🪟 **Windows 환경에 최적화**

---

## 🔧 수정 및 개선 사항

### 1️⃣ 버그 수정

* 기존 Windows 버전에서
  일부 **특수 문자 키 입력 시 사운드가 재생되지 않던 오류 해결**

### 2️⃣ 편의 기능 추가

* 📁 **파일 아이콘 추가**
* 🔇 **원클릭 음소거 버튼**
* 🚀 **Windows 시작 프로그램 등록 버튼**

### 3️⃣ 디자인 개선

* 🎨 **동물의 숲 감성 테마 적용**
* 전체 UI를 보다 아기자기하고 직관적으로 개선

---

## ▶️ 실행 방법

1. `동물의 숲 키보드.exe` 실행
2. 🎉 끝! (설치 과정 없이 바로 사용 가능)
